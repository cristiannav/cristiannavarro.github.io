# Portfolio — Cristian Navarro

Portfolio profesional de Software Engineer con UI moderna (dark/neon), animaciones 3D
inmersivas y un formulario de contacto funcional que envía emails.

- **Frontend:** Vite + React + TypeScript + Tailwind CSS + Three.js (`@react-three/fiber`) + Framer Motion.
- **Backend:** NestJS + Resend para el endpoint `POST /api/contact`

## Requisitos

- Node.js 18+ y npm.
